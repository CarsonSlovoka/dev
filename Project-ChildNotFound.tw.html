Project ChildNotFound.tw

失蹤兒少開放資料平台計畫

What we are going to do

  • 從失蹤兒童少年資料管理中心 http://www.missingkids.org.tw/ 網頁 parse 進資料庫 (各縣市似乎也有分散的網站，也都蒐集進來)
  • 參考國外的活動 404 missing children http://notfound.org/ http://www.childnotfound.com/ 提供台灣版本，讓有心協助的網站主只要嵌入簡單的 script 就能在網站 404 頁面顯示失蹤兒童訊息
  • 也許提供 API 提供資料 (待討論)
